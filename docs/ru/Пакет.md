# Пакет
#### В UPSS общение между сервером и клиентом осуществляется через отправку пакетов(класс Package)
### Структура
- Путь
- Тип(SEND или STATUS)
- Данные(основное тело)

### Инициализация
```python
Package("ПУТЬ", Types.ТИП, data)
```
``data`` или же данные должны быть в формате JSON(пример ``{"a":"b"}``)
